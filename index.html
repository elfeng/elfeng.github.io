WORKSHOP

  • Platform: Web
  • Tools: HTML, CSS, Bootstrap, Git, Brackets

Sign-up page

Feed view

WorkShop aims to provide a platform for local artists to upload and share their creative work with others (e.g. photos, graphic designs). It is a test app developed in one of the Web Teams at UBC Launch Pad as their first project, and serves as a predecessor for WorkDesk. User can create a profile, upload an image file from local with a description, and browse all images posted in the feed.

RESPONSIBILITIES

  • Implemented a responsive UI for the sign-up page, the navbar, and the header section of the landing page (which includes a sign-in form) following a mockup.
  • Learned HTML, CSS, and Bootstrap in 3 weeks.
