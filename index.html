LIBRARY FINDER

  • Platform: Web
  • Tools: GWT, AJAX, JSON, JUnit, Eclipse, Git, HTML, CSS

Login page

Map view

Library Finder was written in Java using the Google Web Toolkit framework. AJAX was used for client-server communication. Users can log in to the web service with a Gmail account. On the web page, there is a map with clickable icons marking library locations for a selected area in B.C. Users can search based on city, add libraries to a favourites list, and go to a library location on the map from either their favourites list or the table below the map.

RESPONSIBILITIES

  • Imported and parsed JSON dataset of BC libraries from DataBC.
  • Wrote and executed JUnit tests for parser.
  • Implemented login using Gmail authentication.
  • Built login page with HTML and CSS
  • Implemented Facebook integration to enable sharing of the web page.

YELP REVIEW BROWSING UI REDESIGN

  • Target platform: Web
  • Tools: PPT, Scrap paper

Demo for mid-fi prototype built with PPT.

Demo for low-fi paper prototype.

In this HCI project, our team of 4 redesigned some of the UI elements of the review browsing section on Yelp to improve its user experience. We went through two iterations in the UI design process. The first iteration involved gathering requirements, designing a conceptual model, developing a paper prototype, and evaluating it with cognitive walkthrough. The second iteration involved developing a semi-functional mid-fi prototype based on feedback from iteration #1, and evaluating it by running a usability study.

Key features:

  • Added search reviews by rating
  • Added search reviews by 'useful' votes
  • Improved visibility of multi-word search functionality by using an example input placeholder and an instructional title for search bar
  • Added one-line summaries/titles for each review to facilitate quick extraction of information
  • Restricted default length of each review to avoid long scrolling and allowed for a 'read more' option instead

RESPONSIBILITIES

  • Conducted usability testing with participants to gather usability and UX data, involving a questionnaire, a structured interview and direction observation.
  • Prepared study materials: questionnaires, interview questions, and coding sheets for observation.
  • Drafted evaluation plan and usability study report.
  • Compiled and edited demo video for paper prototype.
